How It Works

Wave Function Collapse (WFC) is an algorithm inspired by quantum mechanics that generates patterns from local constraints.

The Process:

  1. Superposition: Each cell starts in a superposition of all possible states (tiles)
  2. Observation: The algorithm picks the cell with lowest entropy (uncertainty)
  3. Collapse: That cell collapses to a single state
  4. Propagation: Neighboring cells update their possibilities based on constraints
