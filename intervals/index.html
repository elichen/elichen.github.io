How to Play

Explore musical intervals in three different modes:

  • Identify Interval: Name the interval shown on the staff.
  • Create Interval: Select the correct note to create the given interval.
  • Ear Training: Listen to an interval and identify it.

Click on piano keys or use your computer keyboard to select notes. Earn points for correct answers!
