Each predicted grid point asks the same question: “At this x/y coordinate, what color belongs here?” The answer comes only from the learned parameters.

Is this actually AI?

Yes—just deliberately tiny. The browser initializes a multilayer perceptron, measures its pixel error, differentiates that error through two hidden layers, and updates every parameter with Adam. There is no image API and no pre-trained model.
